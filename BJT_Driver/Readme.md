# BJT LED Driver Circuit (Simulation)

This short tutorial demonstrates how a **Bipolar Junction Transistor (BJT)** can be used as a simple LED driver in a common‑emitter configuration.

## 🔧 Circuit Overview
- **Transistor:** NPN BJT (e.g., 2N2222)
- **Supply Voltage:** 5 V
- **Collector Resistor:** 1 kΩ
- **Base Resistor:** 1 kΩ
- **Load:** LED
- **Base Voltage:** ≈ 460 mV
- **Input Voltage:** 1 V
- **Measured Current:** 3.3 mA RMS

## ⚙️ How It Works
1. The base receives a small input voltage (~1 V), turning the transistor **ON**.  
2. Current flows from collector to emitter, lighting the LED.  
3. The collector resistor limits current to protect both the LED and transistor.  
4. The simulation graph shows current waveform and RMS value.

## 📊 Simulation Notes
- Time scale: 2 ms/div  
- Observe LED brightness changes with base voltage.  
- Ideal for learning transistor switching and current amplification.
