An HTML/JS web server max 2MB hosted on an ESP8266. Uses your public IP as the web site IP, connects to your WiFi. Open the link, press ctrl+f and search for ESP8266_WebServer.zip.

You must download and install the LittleFS plugin and the LittleFS library!

Links are in the video description:

<b>You must have the LittleFS plugin for Arduino IDE installed!</b>
